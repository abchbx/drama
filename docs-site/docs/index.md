## What This Project Does

The Multi-Agent Drama System coordinates a Director agent and multiple Actor agents to produce drama content with:

- Shared state consistency through blackboard pattern
- Strict role-based memory boundaries
- Character-scoped views for each agent
- Long-context control through automated memory folding
- Resilient routing with heartbeat and timeout handling
