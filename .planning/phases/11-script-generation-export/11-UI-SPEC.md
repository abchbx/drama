## Visual Hierarchy

Primary screen focal point: Export Script button (largest interactive element, uses accent color #89b4fa)

Visual priority order:
1. Session selector dropdown (primary action — select what to export)
2. Format radio buttons (secondary action — choose export format)
3. Export Script button (primary CTA — triggers download)
4. Session metadata preview (informational — confirms selection)
