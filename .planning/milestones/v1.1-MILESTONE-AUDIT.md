## Flow Verification

All user journeys through the system are complete:

### Complete Drama Session
```
1. Initialize DramaSession with config and services
2. Director plans plot backbone (uses LlmProvider)
3. Director signals scene start (writes to procedural layer)
4. Router distributes scene start to all actors (broadcast)
5. Round-robin actor turns with timeout fallbacks
6. Director fact-checks and arbitrates
7. Director signals scene end (writes to procedural layer)
8. Optionally promote scenario to core
```

### Error Recovery
- Silent actor → Director fallback continues session
- Actor core write violation → Error logged, session continues
- Token overflow → Semantic layer folds, summary created, session continues
