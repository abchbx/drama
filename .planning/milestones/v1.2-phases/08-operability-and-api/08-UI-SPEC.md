## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #1e1e2e | Background, surfaces |
| Secondary (30%) | #313244 | Cards, sidebar, nav |
| Accent (10%) | #89b4fa | Primary buttons, connection status indicator (connected state) |
| Destructive | #f38ba8 | Error messages |

Accent reserved for: primary CTA buttons, connected status indicator

Additional semantic colors:
- Success: #a6e3a1 (success messages, healthy status)
- Warning: #f9e2af (reconnecting status)
- Text primary: #cdd6f4
- Text muted: #6c7086
- Border: #45475a

Secondary CTAs:
- Start Scene (when session is idle)
